# 2pro's Extension Gallery
These extensions help coders improve their own Scratch/TurboWarp/Penguinmod projects. 
By: 2pro12342. 

## Panel GUI
Panel GUI will help coders/users to create HTML/JS Windows. These windows are Customize-able, Draggable, and Close-able. There are many Elements to create like: buttons, Sliders, Check Boxes, Drop-Downs, Text UI, HTML I-Frame, and more. The Panel GUI extension has Window Management, Elements, Easy Button Detections (Only in V2), Custom Window Styling, Text Fonts, Window ID's, Element ID's, Multi Window Opening, Element Controls, and more.
